Project Submission README
------------------------------------------------------------

The ./code directory contains the -relevant- source files implemented during this project,
including all code needed for the optimization passes and the experimental procedure scripts.
The entire source code collection can be found online at: https://github.com/yln/llvm-pgo

The ./doc directory contains the documents produced to accompany this code, including the project
report, Master Examination questions and answers, and the PGO literature survey.
